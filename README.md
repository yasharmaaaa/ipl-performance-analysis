# 📊 IPL Performance Analysis: Kohli vs Sharma

## 📌 Overview
This project presents a statistical comparison between Virat Kohli and Rohit Sharma based on their IPL performance.

## 🎯 Objectives
- Year-wise performance analysis
- Total runs comparison
- Best season identification

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib

## 📈 Insights
- Kohli has higher peak performance
- Rohit shows consistent performance
